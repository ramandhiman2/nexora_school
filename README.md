# 🎓 Nexora International School Website

A premium modern school website landing page built using **HTML**, **CSS**, **Bootstrap 5**, and **JavaScript**.  
This project features a clean SaaS-style UI inspired by modern Awwwards and Dribbble web designs.

---

# 🚀 Features

- Modern Responsive UI
- Bootstrap 5 Layout
- Premium Hero Section
- Smooth Scroll Animation
- Interactive Navbar
- Animated Cards
- FAQ Accordion
- Testimonials Section
- Academic Programs Showcase
- Modern Footer
- Mobile Responsive Design
